# London (Ontario) – Notes

## What's in this folder

Financial statements for the City of London, Ontario for years 2020-2024.

Each year folder contains:
- **FinancialReport.pdf** - The City of London's Annual Financial Report (comprehensive report with financial analysis, statistics, and consolidated statements)
- **ConsolidatedFinancialStatements.pdf** - Standalone Consolidated Financial Statements

## Coverage

- **2020**: Financial Report + Consolidated Financial Statements
- **2021**: Financial Report + Consolidated Financial Statements
- **2022**: Financial Report + Consolidated Financial Statements
- **2023**: Financial Report + Consolidated Financial Statements
- **2024**: Consolidated Financial Statements only (Financial Report not yet published)

## Source

City of London – Financial Reports page:
https://www.london.ca/government/property-taxes-finance/financial-reports

## Notes

- The 2024 Financial Report has not been published yet on the City's website (as of December 2024)
- The consolidated financial statements are typically published before the full annual financial reports
- All files were successfully downloaded using curl with user-agent headers
